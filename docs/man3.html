orderpack man(3) pages

grouping page description
orderpack:FRACTILEfndnth Return Nth lowest ordered VALUE of array, i.e. return fractile of order N/SIZE(array) (InsertSort-like)
orderpack:FRACTILEindnth Return INDEX of Nth ordered value of array, i.e fractile of order N/SIZE(array) (QuickSort-like)
orderpack:FRACTILEvalnth Return VALUE of Nth lowest ordered value of array, i.e fractile of order N/SIZE(array) (QuickSort-like)
orderpack:MEDIANindmed Returns INDEX of median value of an array.
orderpack:MEDIANmedian Calculates median VALUE of array. If number of data is even, return average of the two "medians".
orderpack:MEDIANvalmed finds the median VALUE of an array
orderpack:MULTIPLICITYmulcnt Give the multiplicity for each array value (number of times that it appears in the array)
orderpack:PERMUTATIONctrper generate a random permutation of an array leaving elements close to initial locations
orderpack:RANK:PARTIAL:UNIQUEunipar partially rank an array removing duplicates
orderpack:RANK:PARTIALrapknr partially ranks an array up to a specified number of values, in DECREASING order.
orderpack:RANK:PARTIALrefpar partially ranks an array up to specified number of elements in ascending order (QuickSort-like)
orderpack:RANK:PARTIALrinpar creates partial rank index of N lowest values in an array
orderpack:RANK:PARTIALrnkpar partially rank array, up to order N (ie. N is the number of sorted elements to return) (QuickSort-like)
orderpack:RANK:UNIQUEuniinv a MergeSort inverse ranking of an array, with duplicate entries assigned the same rank.
orderpack:RANK:UNIQUEunirnk performs a MergeSort ranking of an array, with removal of duplicate entries.
orderpack:RANKmrgref create an INDEX that defines the order of array sorted in ascending order (basic merge-sort)
orderpack:RANKmrgrnk produces an INDEX that sorts an input array (optimized merge-sort)
orderpack:SORT:PARTIALinspar partially sorts an array, bringing the N lowest values to the beginning of the array
orderpack:SORTinssor Sorts array into ascending order (Insertion sort, generally for small or nearly sorted arrays)
orderpack:SORTrefsor Sorts array into ascending order (Quicksort)
orderpack:UNIQUEunista (Stable unique) Removes duplicates from an array otherwise retaining original order
