grouping page description
orderpack:FRACTILEfndnth Return Nth lowest value of an array, i.e. return fractile of order N/SIZE(array) (InsertSort-like)
orderpack:FRACTILEindnth Return INDEX of Nth value of array, i.e fractile of order N/SIZE(array) (QuickSort-like)
orderpack:FRACTILEvalnth Return VALUE of Nth value of array, i.e fractile of order N/SIZE(array) (QuickSort-like)
orderpack:MEDIANindmed Returns index of median value of an array.
orderpack:MEDIANmedian Return median value of array. If even number of data, average of the two "medians".
orderpack:MEDIANvalmed finds the median of an array, averaging the two central values when size of array is even.
orderpack:MULTIPLICITYmulcnt Give the multiplicity for each array value (number of times that it appears in the array)
orderpack:PARTIAL_RANKrapknr Ranks partially XDONT by IRNGT, up to order NORD, in decreasing order. rapknr = (rnkpar backwards)
orderpack:PARTIAL_RANKrefpar partially ranks any array up to specified number of elements
orderpack:PARTIAL_RANKrinpar creates partial rank index of N lowest values in an array
orderpack:PARTIAL_RANKrnkpar Ranks partially XDONT by IRNGT, up to order NORD
orderpack:PARTIAL_RANK_UNIQUEunipar partially rank an array removing duplicates
orderpack:PARTIAL_SORTinspar partially sorts an array, bringing the N lowest values to the beginning of the array
orderpack:PERMUTATIONctrper generate a random permutation of an array leaving elements close to initial locations
orderpack:RANKmrgref produces a sorted ranking of input array (basic merge-sort)
orderpack:RANKmrgrnk produces a sorted ranking of input array (optimized merge-sort)
orderpack:RANK_UNIQUEuniinv a merge-sort inverse ranking of an array, with duplicate entries assigned the same rank.
orderpack:RANK_UNIQUEunirnk performs a Merge-sort ranking of an array, with removal of duplicate entries.
orderpack:SORTinssor Sorts array into ascending order (Insertion sort)
orderpack:SORTrefsor Sorts array into ascending order (Quicksort)
orderpack:UNIQUEunista (Stable unique) Removes duplicates from an array otherwise retaining original order
