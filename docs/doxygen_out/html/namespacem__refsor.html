16 !! Sorts XDONT into ascending order - Quicksort
17 !!__________________________________________________________
18 !! Quicksort chooses a "pivot" in the set, and explores the
19 !! array from both ends, looking for a value > pivot with the
20 !! increasing index, for a value <= pivot with the decreasing
21 !! index, and swapping them when it has found one of each.
22 !! The array is then subdivided in 2 ([3]) subsets:
23 !! { values <= pivot} {pivot} {values > pivot}
24 !! One then call recursively the program to sort each subset.
25 !! When the size of the subarray is small enough, one uses an
26 !! insertion sort that is faster for very small sets.
27 !! Michel Olagnon - Apr. 2000
28 !!__________________________________________________________
29 !!__________________________________________________________
30  Real (kind=kdp), Dimension (:), Intent (InOut) :: xdont
31 ! __________________________________________________________
32 !
33 !
