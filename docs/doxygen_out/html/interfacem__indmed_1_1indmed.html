NAME

indmed(3f) - [orderpack:MEDIAN] Returns index of median value of an array. (LICENSE:CC0-1.0)

SYNOPSIS

 Subroutine ${KIND}_indmed (XDONT, INDM)

   ${TYPE} (kind=${KIND}), Dimension (:), Intent (In) :: XDONT
   Integer, Intent (Out) :: INDM

Where ${TYPE}(kind=${KIND}) may be

   o Real(kind=real32)
   o Real(kind=real64)
   o Integer(kind=int32)

DESCRIPTION

Finds the index of the median of XDONT using the recursive procedure described in Knuth, The Art of Computer Programming, vol. 3, 5.3.3 - This procedure is linear in time, and does not require to be able to interpolate in the set as the one used in INDNTH. It also has better worst case behavior than INDNTH, but is about 30% slower in average for random uniformly distributed values.

OPTIONS

XDONT array to find the median value of. INDM index of the median value.

EXAMPLES

Sample program:

program demo_indmed use M_indmed, only : indmed implicit none !x!call indmed(yyyyyy) end program demo_indmed
