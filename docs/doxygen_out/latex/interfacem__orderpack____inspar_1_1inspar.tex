P\+S\+O\+R\+T(3f) partially sorts I\+N\+O\+U\+T\+V\+A\+LS, bringing the N\+O\+RD lowest values to the beginning of the array.

Internally, this subroutine uses an insertion sort, limiting insertion to the first N\+O\+RD values. It does not use any work array and is faster when N\+O\+RD is very small (2-\/5), but worst case behavior can happen fairly probably (initially inverse sorted). Therefore, in many cases, the refined quicksort method is faster.\hypertarget{interfacem__orderpack____inspar_1_1inspar_autotoc_md56}{}\doxysubsubsection{O\+P\+T\+I\+O\+NS}\label{interfacem__orderpack____inspar_1_1inspar_autotoc_md56}
I\+N\+O\+U\+T\+V\+A\+LS The array to partially sort N\+O\+RD number of sorted values to return.\hypertarget{interfacem__orderpack____inspar_1_1inspar_autotoc_md57}{}\doxysubsubsection{E\+X\+A\+M\+P\+L\+ES}\label{interfacem__orderpack____inspar_1_1inspar_autotoc_md57}
