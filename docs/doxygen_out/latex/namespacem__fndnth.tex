Return N\+O\+R\+Dth value of X\+D\+O\+NT, i.\+e. fractile of order N\+O\+R\+D/\+S\+I\+ZE(X\+D\+O\+NT).

This subroutine uses an insertion sort, limiting insertion to the first N\+O\+RD values. It is faster when N\+O\+RD is very small (2-\/5), and it requires only a work array of size N\+O\+RD and type of X\+D\+O\+NT, but worst case behavior can happen fairly probably (initially inverse sorted).
