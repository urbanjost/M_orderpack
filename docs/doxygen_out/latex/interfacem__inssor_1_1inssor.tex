Sorts X\+D\+O\+NT into ascending order (Insertion sort)

This subroutine uses an insertion sort. It does not use any work array and is faster when X\+D\+O\+NT is of very small size ($<$ 20), or already almost sorted, but worst case behavior can happen fairly probably (initially inverse sorted).
