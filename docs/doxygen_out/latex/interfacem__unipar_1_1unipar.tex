Ranks partially X\+D\+O\+NT by I\+R\+N\+GT, up to order N\+O\+RD at most, removing duplicate entries

This routine uses a pivoting strategy such as the one of finding the median based on the quicksort algorithm, but we skew the pivot choice to try to bring it to N\+O\+RD as quickly as possible. It uses 2 temporary arrays, where it stores the indices of the values smaller than the pivot (I\+L\+O\+WT), and the indices of values larger than the pivot that we might still need later on (I\+H\+I\+GT). It iterates until it can bring the number of values in I\+L\+O\+WT to exactly N\+O\+RD, and then uses an insertion sort to rank this set, since it is supposedly small.
