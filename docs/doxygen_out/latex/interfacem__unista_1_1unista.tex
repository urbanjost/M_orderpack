U\+N\+I\+Q\+U\+E(3f) does a stable removal of duplicates from an array.

It leaves in the initial set only those entries that are unique, packing the array, and leaving the order of the retained values unchanged.

Internally this subroutine uses Merge-\/sort unique inverse ranking.\hypertarget{namespacem__unista_autotoc_md170}{}\doxysubsubsection{O\+P\+T\+I\+O\+NS}\label{namespacem__unista_autotoc_md170}
