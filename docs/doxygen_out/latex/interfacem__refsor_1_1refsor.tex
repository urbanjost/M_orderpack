Sorts I\+N\+O\+U\+T\+V\+A\+LS into ascending order (Quicksort)

This version is not optimized for performance, and is thus not as difficult to read as some other ones.

Internally, This subroutine uses Quick-\/sort in a recursive implementation, and insertion sort for the last steps with small subsets. It does not use any work array.

The Quick-\/sort chooses a \char`\"{}pivot\char`\"{} in the set, and explores the array from both ends, looking for a value $>$ pivot with the increasing index, for a value $<$= pivot with the decreasing index, and swapping them when it has found one of each. The array is then subdivided in two subsets\+: \begin{DoxyVerb}{ values <= pivot} {pivot} {values > pivot}
\end{DoxyVerb}


It then recursively the procedure to sort each subset. When the size of the subarray is small enough, it switches to an insertion sort that is faster for very small sets.\hypertarget{namespacem__refsor_autotoc_md121}{}\doxysubsubsection{O\+P\+T\+I\+O\+NS}\label{namespacem__refsor_autotoc_md121}
