Return N\+O\+R\+Dth value of X\+D\+O\+NT, i.\+e fractile of order N\+O\+R\+D/\+S\+I\+ZE(X\+D\+O\+NT).

This routine uses a pivoting strategy such as the one of finding the median based on the quicksort algorithm, but we skew the pivot choice to try to bring it to N\+O\+RD as fast as possible. It uses 2 temporary arrays, where it stores the indices of the values smaller than the pivot (I\+L\+O\+WT), and the indices of values larger than the pivot that we might still need later on (I\+H\+I\+GT). It iterates until it can bring the number of values in I\+L\+O\+WT to exactly N\+O\+RD, and then finds the maximum of this set.\hypertarget{namespacem__valnth_autotoc_md149}{}\doxysubsubsection{O\+P\+T\+I\+O\+NS}\label{namespacem__valnth_autotoc_md149}
