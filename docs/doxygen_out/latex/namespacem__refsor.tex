Quicksort chooses a \char`\"{}pivot\char`\"{} in the set, and explores the array from both ends, looking for a value $>$ pivot with the increasing index, for a value $<$= pivot with the decreasing index, and swapping them when it has found one of each. \end{DoxyVerb}
