Sorts X\+D\+O\+NT into ascending order using the Quicksort method

Quicksort chooses a \char`\"{}pivot\char`\"{} in the set, and explores the array from both ends, looking for a value $>$ pivot with the increasing index, for a value $<$= pivot with the decreasing index, and swapping them when it has found one of each. The array is then subdivided in 2 (\mbox{[}3\mbox{]}) subsets\+: \begin{DoxyVerb}{ values <= pivot} {pivot} {values > pivot}
\end{DoxyVerb}


One then call recursively the program to sort each subset. When the size of the subarray is small enough, one uses an insertion sort that is faster for very small sets.\hypertarget{namespacem__refsor_autotoc_md91}{}\doxysubsubsection{O\+P\+T\+I\+O\+NS}\label{namespacem__refsor_autotoc_md91}
